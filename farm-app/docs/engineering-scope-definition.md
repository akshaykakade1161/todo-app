# Engineering Scope Definition

## 1. Deliverables
- Fully functional web app with the MVP features outlined in the PRD.
- End‑to‑end test suite.
- CI/CD pipelines.
- In‑port documentation and user guide.

## 2. Exclusions
- Mobile native app (handled in later phase).
- Offline mode / local sync.
- Multi‑tenant deployment.

## 3. Constraints
- Must be built on the Next.js / Node.js / PostgreSQL stack.
- All code must pass our linting and security checks.
- Data privacy: only minimal user data collected.
